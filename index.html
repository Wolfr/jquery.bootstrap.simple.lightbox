Simple jQuery lightbox

Why do all lightboxes out there use a gazillion images and options when all I want is to show a larger image when I click a small one?

Features: keyboard close with Esc or X button; automatically centers in all browsers; easy to add, just add a class="lightbox" to a linked image

Demo

Click an image:

Browser support

Tested in IE6-9, Opera 10+, FF3.5+, Safari 5, Chrome 6

Files

  • reset.css includes a CSS reset; you'll probably already use one in your project.
  • wolfmodal.css includes CSS to make the modal look nice, but this CSS is not required. You can ignore the file and roll your own
  • jquery.wolfmodal.js: the script itself
  • button_close.png button image; the only image used (.gif for IE6)

Usage

  • Make sure you required jquery.wolfmodal.js.
  • Link a small image to a big image: <a href="image_big.jpg"><img src="image_small.jpg" alt="" /></a>
  • Add a class="lightbox" to the <a>
  • Optional: Add alt text to the <img> for the caption
